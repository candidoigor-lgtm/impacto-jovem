## 🚀 Sobre o Projeto
O projeto **Impacto Jovem** é estruturado de forma independente entre camadas de **Backend** e **Frontend**. nosso objetivo é ajudar familiares ou o proprio uuario a sair das drogas os guiando para diminuir a dose e ajuar em momemntos de crise
---

## 🛠️ Tecnologias Utilizadas
Uma combinação moderna de ferramentas robustas para processamento e interfaces responsivas:

* 🐍 **Python (Flask)** — Construção das APIs e regras de negócio do backend.
* 🌐 **HTML5 / CSS3** — Estruturação e estilização visual das interfaces do usuário.
* 🗄️ **Banco de Dados SQL** — Persistência estruturada e segurança dos dados consumidos.

---

## 📂 Estrutura do Repositório
A árvore de diretórios foi planejada para isolar responsabilidades e facilitar o deploy:

```text
├── 📂 backend      # Módulo do servidor Flask e regras do Chatbot
├── 📂 frontend     # Páginas Web, estilos visuais e Dashboard
└── 📄 README.md    # Documentação principal do projeto
```
